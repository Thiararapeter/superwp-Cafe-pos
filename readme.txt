A modern point of sale (POS) system for cafes and restaurants integrated with WooCommerce.

== Description ==

SuperWP Cafe POS is a comprehensive point of sale solution designed specifically for cafes and restaurants using WordPress and WooCommerce. It provides an intuitive interface for managing orders, processing payments, and tracking inventory in real-time.

= Key Features =

* Modern and responsive POS interface
* Real-time product search with suggestions
* Quick category filtering
* Easy cart management
* Multiple payment methods support (Cash, M-PESA)
* Waiter assignment system
* Offline capability
* Receipt printing
* Sales tracking and reporting
* Inventory sync with WooCommerce

= Requirements =

* WordPress 5.6 or higher
* WooCommerce 5.0 or higher
* PHP 7.4 or higher
* Modern web browser with JavaScript enabled

== Installation ==

1. Upload the `superwp-cafe-pos` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin settings under 'POS Settings' in the admin menu
4. Assign POS access to appropriate user roles
5. Access the POS terminal from the admin menu

For detailed documentation and support, please visit:
https://github.com/Thiararapeter/superwp-Cafe-pos

== Frequently Asked Questions ==

= Can I use this without WooCommerce? =

No, SuperWP Cafe POS requires WooCommerce to be installed and activated as it uses WooCommerce's product and order management system.

= Does it work offline? =

Yes, the POS system includes basic offline functionality to continue processing orders during internet connectivity issues.

= Can I customize the receipt format? =

Yes, receipt templates can be customized through the plugin settings.
